# Personal Portfolio Website

A responsive biography and portfolio website built with HTML and CSS.

## Overview
This is a clean, modern portfolio website designed to showcase skills, projects, and contact information. It features a responsive design that works well on both desktop and mobile devices.

## Features
- Responsive Design: Adapts to different screen sizes
- Modern UI/UX: Clean layout with gradient headers and card-based sections
- Skills Showcase: Visual display of technical skills with icons and levels
- Projects Section: Highlights key projects with GitHub links
- Social Integration: Links to social media platforms

## Technologies Used
- HTML5: Semantic structure
- CSS3: Flexbox, Grid, animations, and responsive design
- Font Awesome: Icons for skills and social links

## File Structure
```
Personal-Portfolio/
├── index.html          # Main HTML file

├── img.jpg             # Profile image

```

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Basic text editor (VS Code, Sublime Text, etc.)

### Installation
1. Clone or download the repository
2. Ensure all files are in the same directory:
   - index.html
   - style.css
   - img.jpg (profile image)
3. Open index.html in your browser

### Customization
1. Profile Information:
   - Update name, title, and bio in the index.html file
   - Replace img.jpg with your own profile image

2. Skills:
   - Modify skill cards in the Skills section
   - Update skill levels and descriptions

3. Projects:
   - Replace placeholder project information
   - Update GitHub links

4. Contact Information:
   - Update social media links in the footer
   - Replace placeholder URLs

5. Styling:
   - Modify colors in the style section or style.css
   - Adjust padding, margins, and fonts as needed
